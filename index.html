Speed Racer



Classic Mode

  • This is the vanilla version
  • The timer starts only when a key is pressed on the next screen
  • The game is over once the player finishes the paragraph

Hard Mode

  • This is Hard-Mode
  • The timer starts only when a key is pressed on the next screen
  • The words in the paragraph would disappear and re-appear at intervals
  • The game is over once the player finishes the paragraph
